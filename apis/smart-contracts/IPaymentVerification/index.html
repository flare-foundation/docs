    IPaymentVerification#

    Source

    Interface for verifying Payment attestations within the State Connector. This interface can be utilized at the end of the attestation request process to verify that the data returned by an attestation provider matches the on-chain Merkle proof.

    Functions#

    verifyPayment#

    Defined in IPaymentVerification (Docs, Source).

    function verifyPayment(
    struct Payment.Proof _proof
) external view returns (
    bool _proved);

    Verifies the Payment attestation using a Merkle proof. It checks whether the provided proof corresponds to the on-chain Merkle root for the voting round specified inside the proof.

    Parameters Type Description
    _proof struct Payment.Proof The Payment attestation proof, which includes the Merkle proof and the attestation data. This proof is obtained directly from attestation providers. To learn about the format of this data, see Attestation types.
    Returns Type Description
    _proved bool Whether the attestation is successfully verified.
